# MeuPrimeiroClique-
O projeto "Ampliar o Acesso a Serviços Públicos Digitais" consiste em um site com tutoriais ilustrados que ensinam funções básicas do celular, como tirar fotos e instalar aplicativos, além de orientar os usuários sobre como criar e acessar a conta GOV.br, navegar pela plataforma e emitir documentos digitais de forma simples e acessível.
